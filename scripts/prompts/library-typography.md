# Typography Reference Library

## Font Pairing Recipes

Each recipe pairs a display/heading font with a body font. Choose one recipe and commit to it.

**Editorial Warmth:** Playfair Display (headings) + Source Sans 3 (body). Classic newspaper feel, high contrast between thick/thin strokes. Works for: storytelling, long-form, editorial.

**Modern Precision:** Inter (headings, bold weight) + Inter (body, regular weight). One family, weight contrast only. Works for: clean, professional, tech-forward.

**Humanist Contrast:** Fraunces (headings) + Outfit (body). Soft, warm serif meets geometric sans. Works for: approachable, creative, warm.

**Brutalist Mono:** Space Grotesk (headings) + JetBrains Mono (body). Technical, raw, code-adjacent. Works for: developer portfolios, data-heavy, bold.

**Elegant Serif:** Cormorant Garamond (headings) + Lora (body). Both serifs, but contrasting — display vs text. Works for: luxury, refined, literary.

**Geometric Clean:** DM Sans (headings) + DM Sans (body, lighter weight). Geometric, friendly, modern. Works for: startups, products, minimal.

**Bold Statement:** Syne (headings) + Work Sans (body). Distinctive display font with neutral body. Works for: creative agencies, bold brands, artistic.

**Swiss Precision:** Helvetica Neue or Switzer (headings) + IBM Plex Sans (body). Neo-grotesque meets humanist. Works for: corporate, polished, structured.

## Typographic Scale

Use a mathematical scale ratio. Pick ONE and apply consistently:

| Ratio | Name | Sizes (base 16px) |
|-------|------|-------------------|
| 1.200 | Minor Third | 11, 13, 16, 19, 23, 28, 33 |
| 1.250 | Major Third | 10, 13, 16, 20, 25, 31, 39 |
| 1.333 | Perfect Fourth | 9, 12, 16, 21, 28, 37, 50 |
| 1.414 | Augmented Fourth | 8, 11, 16, 23, 32, 45, 64 |
| 1.500 | Perfect Fifth | 7, 11, 16, 24, 36, 54, 81 |

**Major Third (1.250)** is the safest default — enough contrast without extremes.
**Perfect Fourth (1.333)** adds more drama — good for editorial layouts.
**Perfect Fifth (1.500)** is bold — only use with sparse, spacious layouts.

## Type Rules

- **Never use more than 2 font families.** If you need variety, use weight/style contrast within one family.
- **Heading sizes must be noticeably larger than body.** If the ratio is less than 1.2x, it looks like a mistake.
- **Body text: 16-18px, line-height 1.5-1.7.** This is non-negotiable for readability.
- **Heading line-height: 1.0-1.2.** Tighter than body. Headings should feel compact.
- **Letter-spacing: negative for large headings (-0.02em), positive for small caps/labels (+0.05-0.1em).**
- **Max 4 distinct text sizes on a page.** More than that creates noise, not hierarchy.
